# Example Commands

## Tambah Tugas

```text
ingetin tugas ASD dikumpul Jumat jam 8 malam
```

```text
catat laporan praktikum sistem operasi deadline besok jam 23.59
```

```text
ada kuis matematika diskrit Senin jam 10 pagi
```

## Tambah Jadwal Kuliah

```text
jadwal kuliah ASD setiap Senin jam 08.00 sampai 09.40 di GKU 101
```

```text
jadwal sistem operasi tiap Selasa jam 13.00-14.40 di Lab Komputer
```

## Lihat Tugas

```text
/today
```

```text
/tomorrow
```

```text
/week
```

Atau bisa dengan bahasa bebas:

```text
tampilkan tugas hari ini
```

```text
apa aja tugas minggu ini?
```

## Memory Preferensi

```text
aku lebih suka diingatkan H-1 dan 2 jam sebelum deadline
```

```text
ingat bahwa tugas besar harus diingatkan mulai H-3
```

## Edit / Hapus / Selesai

```text
hapus tugas ASD
```

```text
tugas matematika diskrit sudah selesai
```

```text
ubah deadline laporan SO jadi Jumat jam 9 malam
```
